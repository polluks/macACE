#  Hnefataf on the Amiga

This project brings the medieval Norse boardgame to the Amiga platform for the first time.

This version follows the Copenhagen Hnefatafl Ruleset.

The current release version is two player only, while a game AI is in production. 

This Game uses the [Amiga C Engine](https://github.com/AmigaPorts/ACE) and the help of KaiN in particular was and is a great help.
