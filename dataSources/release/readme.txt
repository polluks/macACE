Hnefatalf is a medieval Norse board game that survived regular game play until the late 18th century. This Amiga version follows the Copenhagen rules of the game (a rulebook is attached).

This version of the game is version 1.0 which does not contain any A.I or single player mode. 

Brown are attackers and White are defenders.

Controls

To move a piece, select a piece of the team that is in play using your mouse, then click the square you wish to move too.

Invalid moves will just not do anything, so check whose turn it’s meant to be!

There are two Key controls

ESC- Quit the game
Hitting escape will quit the game and return to workbench or the ROM OS.

R- Resign
Pushing R will resign you from the game.

Bugs
There are some small quirks that are already known.

-Selecting Squares
I have found in play throughs on some A500 emulation configs that the selecting of the squares can be quite strict, needing to have the pinch of the fingers be in the exact middle.

-Key Inputs not working
I have found that sometimes key inputs for the game don’t always work. If you’re in the game proper you need to select a piece first and then key input will generally work.

If you’re in the game end screen, sometimes just mashing the key you want works.

Bug reporting can be done as submitting a issue to the git repo or @ing Jose Ambraelle in the AmigaGameDev discord.
Game Rules

The only game rule not implemented is Perpetual repetition. Which means if you are stuck in a loop of repeating moves white automatically looses. There are some exceptions which are detailed in the game rules.
